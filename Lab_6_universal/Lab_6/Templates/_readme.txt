Формат входного файла (<...> - комментарии, \n - перенос строки. ДАННЫЕ РАЗДЕЛЕНЫ ПРОБЕЛАМИ):

<ВНИМАНИЕ! На данный момент формат комментарии НЕ ПОДДЕРЖИВАЕТ!>
<bof>
x0 y0 z0 \n <координаты центра объекта>
nPoints \n <количество точек объекта>
x1 y1 z1 \n
x2 y2 z2 \n
<...>
xN yN zN \n <координаты точек>
0 1 ... 1 \n
1 0 ... 0 \n
......... \n
1 1 ... 0 \n
<Матрица смежности каркаса (полная). 1 - есть ребро, 0 - нет ребра. Размерность - nPoints x nPoints>
<eof>
